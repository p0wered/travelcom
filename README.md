# Travelcom App

## Project setup

* ### `Donwload project`
* ### `npm install`
* ### `npx expo start`

## Build Android or iOS file

* ### `Login to your EAS account - eas login`
* ### `Configure the project - eas build:configure`
* ### `Configure the eas.json`
* ### `Build for your platform - eas build --platform <your platform>`
